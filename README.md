# AWS Troubleshooting Labs

Hands-on troubleshooting scenarios commonly encountered by AWS Cloud Support Engineers.  

Each lab includes:
- Problem description
- Tools used (AWS CLI, CloudWatch, VPC Flow Logs)
- Root cause analysis
- Resolution steps
- Prevention recommendations

This repository demonstrates practical troubleshooting skills aligned with AWS Support workflows.
